absolute_path for rel_path

相対パスを絶対パスに変換します。

引数に与えられた相対パスを、 make_with もしくは、set_base_path で与えられている基準パスをもとにして、絶対パスに変換します。

引数に与える相対パスの形式は、基準パスと同じ形式（POSIX 形式もしくは HFS 形式）である必要があります。

Parameters
  • rel_path (Unicode text) : 相対パス
Result

Unicode text : 絶対パス
